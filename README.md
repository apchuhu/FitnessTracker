# Fitness Tracker - MotionFlow

## What MotionFlow Does?
>MotionFlow is an Android fitness tracker designed to help users monitor daily step counts, log workouts, set personal fitness goals, and track long-term progress.
>The app uses Firebase for real-time data storage and secure user authentication, allowing users to access their fitness data across sessions.

## Project Purpose
>MotionFlow is designed to demonstrate the development of a native Android application that integrates real-time data storage, authentication, and user-driven features.
>The project applies mobile development concepts to solve a real-world problem, tracking and visualizing personal fitness progress over time.

## How to Install and Run
1. Install Android Studio
2. Ensure that you have an emulated Android device
3. Clone the project repository and open it in Android Studio
4. Add a new virtual device/emulator to Android Studio (Recommended Pixel 5)
5. Once the app is opened, go to Tools > Firebase > Realtime Database > Get started with Realtime Database [Java]
6. Run the app, if you encounter any gradle sync issues sync your gradle version with the project
